# Este repo contiene 4 archivos:
    - fibonacci.py: Son funciones que cálculan el número fibonacci n veces.
    - sumatoria.py: Funciones que suman todos los enteros del 0 al n.
    - factorial.py: Funciones que calculan el factorial de n (n!).
    - buscardatos.py: Una función que analiza una base de datos, y compara  
        los datos ingresado con los de la base de datos, devolviendo un key que
        indica cual es el usuario con los datos ingresados. Si este usuario   
        no está en la BD se retorna un False
